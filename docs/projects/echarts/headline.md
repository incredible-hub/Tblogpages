## 可视化面板介绍

应对现在数据可视化的趋势，越来越多企业需要在诸如营销数据、生产数据、用户数据等场景下使用可视化图表来展示数据，让数据更加直观，数据特点更加突出。

## 项目技术

- div + css 布局
- flex 布局
- Less
- 原生 js + jquery 使用
- rem 适配
- Echarts 基础
